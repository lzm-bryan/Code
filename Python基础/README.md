# Python 基础学习笔记

Python 常用数据结构和语法学习笔记。

## 内容

| 文件 | 主题 | 内容 |
|------|------|------|
| Hash.ipynb | 哈希表 | dict、defaultdict、set、两数之和 |
| Deque.ipynb | 队列 | deque、Queue、PriorityQueue、异步队列 |

## 常用数据结构复杂度

| 操作 | list | dict/set | deque |
|------|------|----------|-------|
| 索引访问 | O(1) | O(1) | O(1) |
| 头部插入 | O(n) | - | O(1) |
| 尾部插入 | O(1) | O(1) | O(1) |
| 查找 | O(n) | O(1) | O(n) |
